Deskripsi Project
Perkembangan teknologi informasi mendorong berbagai usaha untuk mengelola data secara digital. Pada studi kasus ini dibuat sebuah API Toko Sepeda yang berfungsi untuk mengelola data produk sepeda seperti:

Menambahkan data sepeda
- Melihat daftar sepeda
- Melihat detail sepeda
- Mengubah data sepeda
- Menghapus data sepeda

API ini dibuat menggunakan Laravel dan diuji menggunakan Postman.

Teknologi yang Digunakan

- Laravel
- PHP
- REST API
- Postman
- JSON
